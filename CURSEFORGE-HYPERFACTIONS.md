## Description

```
HyperFactions is the complete faction management solution for Hytale servers.

Create factions. Claim territory. Forge alliances. Wage war. All through an intuitive GUI or powerful commands.

⚠️ EARLY ACCESS: Actively developing. Report bugs and shape the future of Hytale factions!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

WHY HYPERFACTIONS?

Built from the ground up for Hytale - not a port, not a clone. Modern architecture with features server owners have been asking for:

• Full GUI management - no command memorization required
• Strategic power system that rewards active play
• Automatic backups with smart rotation
• Permission-based limits (claims, power) per group
• 11 configurable territory flags per faction
• Multi-chunk zone system for admins

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

CORE FEATURES

FACTION MANAGEMENT
• Create factions with custom names, colors, and tags
• Three-tier hierarchy: Leader → Officer → Member
• Smart invitation system with expiration
• Activity logging for accountability
• Open/closed membership modes

TERRITORY SYSTEM
• Chunk-based claiming with visual map
• Power-based claim limits
• Configurable adjacent-only claiming
• Inactive claim decay system
• Multi-world support with whitelist/blacklist

POWER SYSTEM
• Personal power per player
• Automatic regeneration over time
• Death penalties create strategic risk
• Overclaiming when enemies are weak
• Permission-based power limits for VIPs

DIPLOMACY
• Form alliances with mutual agreement
• Declare enemies for full warfare
• Relation-based PvP rules
• Alliance chat for coordination

TERRITORY PERMISSIONS
Factions control 11 flags in their territory:
• PvP and friendly fire
• Building and container access
• Item drop/pickup
• Mob spawning and damage
• Hunger and fall damage

Server admins can lock or force any flag server-wide.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ADMIN TOOLS

ZONE SYSTEM
Create admin-managed SafeZones and WarZones:
• Named zones spanning multiple chunks
• Radius claiming (/f admin zone radius spawn 5 circle)
• 11 configurable flags per zone
• Perfect for spawn, arenas, and events

BACKUP SYSTEM
Enterprise-grade data protection:
• Automatic hourly/daily/weekly backups
• GFS rotation policy
• Manual backup creation
• One-command restore

ADDITIONAL TOOLS
• In-game update system
• Debug mode for troubleshooting
• Force disband/modify any faction
• Bypass mode for investigations

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

QUICK START

1. Drop HyperFactions.jar in your mods folder
2. Start your server
3. Grant permissions (HyperPerms recommended):
   /hp group setperm default hyperfactions.use
   /hp group setperm default hyperfactions.faction.*
   /hp group setperm default hyperfactions.member.*
   /hp group setperm default hyperfactions.territory.*
   /hp group setperm default hyperfactions.teleport.*
   /hp group setperm default hyperfactions.relation.*
   /hp group setperm default hyperfactions.chat.*
   /hp group setperm default hyperfactions.info.*
4. Create your first faction: /f create MyFaction

Full GUI available: just type /f

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

COMMAND OVERVIEW

PLAYERS
/f create <name> - Create a faction
/f invite <player> - Invite a player
/f claim - Claim current chunk
/f map - View territory map
/f home - Teleport to faction home
/f ally/enemy <faction> - Manage relations
/f c <message> - Faction chat
/f stuck - Escape hostile territory

ADMINS
/f admin - Open admin GUI
/f admin zone create <name> <safe|war> - Create zone
/f admin backup create - Manual backup
/f admin reload - Reload config

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

INTEGRATIONS

HYPERPERMS (Recommended)
Full permission integration with category-based nodes:
• hyperfactions.faction.* - Faction management
• hyperfactions.member.* - Member management
• hyperfactions.territory.* - Land claiming
• hyperfactions.admin.* - Administration

Permission-based limits:
• hyperfactions.limit.claims.50 - Max 50 claims
• hyperfactions.limit.power.100 - Max 100 power

Get HyperPerms: https://www.curseforge.com/hytale/mods/hyperperms

CHAT INTEGRATION
• Faction tags in public chat
• Configurable tag format and colors
• Relation-based coloring (allies green, enemies red)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

CONFIGURATION

Fully configurable via config.json:
• Power balance (max power, regen rate, death penalty)
• Claim limits and decay settings
• Teleport warmup and cooldown
• Combat tag duration
• Backup retention policies
• Chat formatting
• And much more...

See full documentation for all 50+ configuration options.
```
