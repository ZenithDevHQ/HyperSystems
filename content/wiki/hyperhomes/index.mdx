# Welcome to HyperHomes

HyperHomes is a comprehensive home management plugin for Hytale servers. With GUI-based management, home sharing, and seamless cross-world teleportation, your players can easily manage their homes.

## Features

- **Multiple Named Homes** - Allow players to set multiple homes with custom names
- **GUI Management** - Intuitive graphical interface for managing homes
- **Home Sharing** - Players can share their homes with friends
- **Warmup & Cooldown** - Configurable delays to balance gameplay
- **Cross-World Teleportation** - Teleport between worlds seamlessly
- **Safe Teleport** - Automatically finds safe landing spots
- **HyperPerms Integration** - Works seamlessly with HyperPerms

## Commands Overview

| Command | Description |
|---------|-------------|
| `/home [name]` | Teleport to a home |
| `/sethome [name]` | Set a new home |
| `/delhome <name>` | Delete a home |
| `/homes` | Open the homes GUI |
