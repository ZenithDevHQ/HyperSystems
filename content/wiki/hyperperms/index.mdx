# Welcome to HyperPerms

HyperPerms is a modern, feature-rich permissions system designed for Hytale servers. With a visual web editor, context-aware permissions, and seamless group inheritance, managing your server's permissions has never been easier.

## Features

- **Visual Web Editor** - Manage permissions through an intuitive web interface
- **Context-Aware Permissions** - Set permissions based on world, region, biome, gamemode, or time
- **Group Inheritance** - Create hierarchical permission groups with weight-based priority
- **Timed Permissions** - Grant temporary permissions that automatically expire
- **Promotion Tracks** - Define clear paths for player progression
- **High-Performance** - LRU caching ensures lightning-fast permission checks
